O que é HTML5 ?

HTML5, sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web. Ela traz novas funções como as tags <video>, <audio> e elemento <canvas> e será o novo padrão para HTML, XHTML e HTML DOM. Com o HTML5, por exemplo, é possível ver vídeos do YouTube sem o Flash.

Vantagens

  • Reduz travamentos em dispositivos móveis.
  • Aumenta a confiança em sites móveis. Quando um usuário quiser usar os serviços da empresa, mas não quiser fazer o download de seu aplicativo, eles podem simplesmente fazer logon no site responsivo da empresa para fazê-lo.
  • A capacidade de suportar elementos nativos de áudio e vídeo sem plugins adicionais.

Desvantagens

  • Ele não pode produzir saída dinâmica sozinho, pois é uma linguagem estática.
  • Às vezes, é difícil entender a estruturação de documentos HTML.
  • Os recursos de segurança oferecidos pelo HTML são limitados.
